Prerequisites
- Host Machine
  - Windows / macOS / Linux
  - Docker Desktop installed and running

Setup docker-compose.yaml in VS Code

Start the Docker Containers
docker-compose up -d
This will start two containers:
- MySQL Database Container
- Hadoop Container

Verify Hadoop & MapReduce
docker exec -it hadoop bash

Check HDFS & YARN
hdfs dfs -ls /
yarn node -list

Run MapReduce test (MANDATORY)
hadoop jar /hadoop-2.8.2/share/hadoop/mapreduce/hadoop-mapreduce-examples-2.8.2.jar pi 1 1

exit from hadoop container:
exit

Download Sqoop
https://archive.apache.org/dist/sqoop/1.4.7/sqoop-1.4.7.bin__hadoop-2.6.0.tar.gz

In mac/linux:
wget https://archive.apache.org/dist/sqoop/1.4.7/sqoop-1.4.7.bin__hadoop-2.6.0.tar.gz

Copy into container:
docker cp sqoop-1.4.7.bin__hadoop-2.6.0.tar.gz hadoop:/opt/


Download MySQL JDBC Driver:
https://repo1.maven.org/maven2/mysql/mysql-connector-java/8.0.28/mysql-connector-java-8.0.28.jar

In mac:
wget https://repo1.maven.org/maven2/mysql/mysql-connector-java/8.0.28/mysql-connector-java-8.0.28.jar

Copy into container:
docker cp mysql-connector-java-8.0.28.jar hadoop:/opt/


docker exec -it hadoop bash
cd /opt
tar -xvzf sqoop-1.4.7.bin__hadoop-2.6.0.tar.gz
mv sqoop-1.4.7.bin__hadoop-2.6.0 sqoop


Set environment variables:
export SQOOP_HOME=/opt/sqoop
export PATH=$PATH:$SQOOP_HOME/bin

Persist:
echo 'export SQOOP_HOME=/opt/sqoop' >> ~/.bashrc
echo 'export PATH=$PATH:$SQOOP_HOME/bin' >> ~/.bashrc
source ~/.bashrc

Verify:
sqoop version


Place in Sqoop lib:
cp /opt/mysql-connector-java-8.0.28.jar $SQOOP_HOME/lib/

Verify:
ls $SQOOP_HOME/lib/ | grep mysql


Verify MySQL Connectivity:
sqoop list-databases \
--connect jdbc:mysql://mysql:3306/ \
--username root \
--password root
